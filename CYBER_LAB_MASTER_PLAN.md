**"Why the Old Ways Fail."**

The classic method (ARP Replay) is effective but can be slow or detected. We will add a **"Modern Warfare"** toggle to the app.

### 1. The "Deauth" Strike (Faster Handshakes)
Instead of waiting 30 minutes (`sleep 1800`), we force the device to reconnect.
*   **Old Way:** Wait for traffic.

*   **Result:** Captures the WPA2 Handshake in **seconds**, not minutes.

### 2. PMKID Attack (Client-less Attack)
*   **Concept:** Attack the router directly without waiting for a user to connect.
*   **Tool:** `hcxdumptool` (Modern replacement for simple airodump).
*   **Benefit:** Works even if no one is home.

### 3. GPU Acceleration (Hashcat)
*   **Concept:** `aircrack-ng` uses the CPU (slow).
*   **Upgrade:** The app will export the capture file to a format readable by **Hashcat** to use the Graphics Card (NVIDIA/AMD).
*   **Speed:** 1000x faster password testing.

## üåê Module 5: Protocol Mastery (The Universal Key)
**"From Ancient Ruins to Future Fortresses."**

We must ensure the tool understands ALL protocols, not just WPA2.

### 1. The Legacy: WEP & WPA1
*   **Status:** Obsolete but dangerous if found.
*   **Attack:** IV (Initialization Vector) Statistical Attack.
*   **Tool:** `aircrack-ng` (PTW method).
*   **Lesson:** "Why we stopped using this in 2004."

### 2. The Standard: WPA2 (CCMP)
*   **Status:** Most common.
*   **Attack:** 4-Way Handshake Capture + Dictionary Attack.
*   **Enhancement:** **WPS Pixie Dust Attack**.
    *   Many WPA2 routers have WPS enabled.
    *   **Tool:** `reaver` or `bully` (integrated into our script).
    *   **Result:** Cracks the PIN in seconds, revealing the password instantly.

### 3. The Fortress: WPA3 (SAE)
*   **Status:** The new standard (2025+).
*   **Defense:** Uses "Simultaneous Authentication of Equals" (Dragonfly).
*   **Attack Vector:** **Downgrade Attack**.
    *   Force the router to accept WPA2 connections if "Transition Mode" is on.
*   **Lesson:** "Why WPA3 is (almost) unhackable via dictionary attacks."
